# Mac OS X 快捷键

以下是可能会出现在 OS X 菜单中的修饰键符号：

+ ⌘	Command 键
+ ⌃	Control键 控制代码
+ ⌥	Option 键
+ ⇧	Shift 键
+ ⇪	Caps Lock
+ Fn 功能键

#### 启动快捷键

按下按键或组合键，直到所需的功能出现/显示（例如，在启动过程中按住 Option 键直到出现“启动管理器”）。注：如果启动功能不起作用，且您使用的是第三方键盘，请连接 Apple 键盘再试一次。
